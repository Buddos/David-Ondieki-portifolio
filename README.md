# David Ondieki — Portfolio

This is a simple static portfolio site built with plain HTML, CSS and a small JavaScript helper for shared header/footer.

Deploy to Vercel (quick):

1. Install Vercel CLI (optional):

```bash
npm i -g vercel
```

2. From the project root run:

```bash
vercel --prod
```

Vercel will detect this as a static project.

Notes:
- Place `David-Ondieki-CV.pdf` in the project root.
- Replace `images/profile-*.jpg`/`.webp` with optimized photos if needed.
